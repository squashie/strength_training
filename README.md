# Home Gym — 7-Week Strength Program

A clean, single-page workout site built for GitHub Pages. No frameworks, no build step — just drop the files in a repo and enable Pages.

## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `home-gym-program`)
2. Upload all four files:
   - `index.html`
   - `exercises.js`
   - `program.js`
   - `app.js`
3. Go to **Settings → Pages**
4. Under "Source", select **Deploy from a branch**
5. Choose `main` branch, `/ (root)` folder
6. Click Save — your site will be live at `https://yourusername.github.io/home-gym-program/`

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main page, styles, layout, HTML structure |
| `exercises.js` | SVG stick-figure illustrations for every exercise |
| `program.js` | All workout data and coaching tips |
| `app.js` | Rendering logic, navigation, modal |

## Features

- Browse by week (1–2, 3–4, 5–6, deload)
- Click any exercise to see its SVG illustration + coaching cues
- Fully responsive (mobile-friendly)
- No internet connection required after first load (no external JS deps)

## Program Overview

3 days/week · 45–60 min · 7-week progressive cycle  
Goal: fat loss + strength · Intermediate level  
Equipment: squat rack with lat pulldown, adjustable bench, barbell + plates, EZ-curl bar
